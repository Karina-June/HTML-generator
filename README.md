## Installations
To install inquirer

```bash
node i inquirer@8.2.4.
```
To install jest
```bash
npm install --save-dev jest
```
To install coverage
```bash
npm run coverage
```
To install template helpers
```bash 
npm install --save template-helpers
```
Application will be invoked using
```bash
node index.js
````
